    • globalmaptiles.GlobalMercator: TMS Global Mercator Profile

  • webservice.redirect_handler: For all redirects from social media logins for user authentication
  • webservice.twitter_app_only_authorisation: Application only authorisation method for twitter
  • webservice.user_authorisation: Authenticates a user to use the social media sites as authenticated by their profile
