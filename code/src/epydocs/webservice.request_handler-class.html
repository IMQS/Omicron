GET(self)

source code 

Handles GET requests received from users.

Parameters:
  • self (request_handler) - Pointer to the current request_handler instance.
  • platforms (String) - List of underscore(_) separated values that indicate which social media to use.
  • tags (String) - List of Underscore(_) separated values that is used for the search tags.
  • function (String) - Which processing function to apply after the data has been collected.
  • location_type (String) - Either area or radius.
  • location (String) - If location_type has the value "area" then location must be in the following format "country state city" else if location_type has the value "radius" then location must be in the following format "longitude_latitude_<radius>"
Returns: JSON Object
This will vary between request depending on what processing functions are applied to the social data that was mined.
