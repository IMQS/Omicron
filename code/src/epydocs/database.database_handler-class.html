get_social_data(self, time_start=None, time_end=None, query=None, database_name="omicron", collection_name=None)

source code 

Finds all the entries from time_start to time_end inclusive that has the same query value as the parameter query. Note That time_start and time_end can both take a value "'all'". The "'all'" value instructs the search to have no upper or lower bounds or both.

Parameters:
  • self (database_handler) - Pointer to the current object.
  • time_start (datetime.datetime) - The time interval the search should start at.
  • time_end (datetime.datetime) - The time interval the search should end at.
  • query (string) - The parameters used for what to search. The string must be formated the same as the parameters of a GET command including the ?.
  • database_name (string) - The name of the database.
  • collection_name (string @raise AttributeError @raise Exception("Connection failure")) - The name of the collection you want to access.
Returns: JSON Object
The data that meet the criteria of the search parameters.
