_available_social_media(self, names)

source code 

Checks to see if the selected social API's are available. If not then a "NotImplementedError" exception will be raised, otherwise it will return a list of social_platform objects.

Parameters:
  • self (Gateway.) - A pointer to the current object.
  • names (List of strings.) - List of names.

GET(self)

source code 

Only for handling reST calls to the gateway.

Cycles through each variable in the platform list and executes a search with the given keywords (tags) on each platform. Parameter platforms and search_tags are encoded into the GET call.

Parameters:
  • platforms (String) - List of social media platforms.
  • search_tags (String) - List of key words that will be used in the search.
  • self (gateway) - a Pointer to the current object.
Returns: GeoJSON array
The raw data that was fetched from the social media API's.

POST(self)

source code 

This function is only for handling rest calls to /gateway. Cycles through each variable in the platform list and executes a search with the given keywords on each platform. Parameter platforms and search_tags are encoded into a JSON Object that will be received from a POST call.

Parameters:
  • Data (JSON Object) - List of social media platforms.
  • self (gateway) - a Pointer to the current object.
Returns: GeoJSON array
The raw data that was fetched from the social media API's.
