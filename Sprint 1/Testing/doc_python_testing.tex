\section{Python Testing}
Here follows a summary of sorts. For more information and complete examples please visit \verb5http://docs.python.org/2/library/unittest.html5.

\section{Misc things that are important}
\begin{itemize}
\item{\textbf{Help} can be found by running - \verb5 python -m unittest -h5}
\item{include \verb! suite = unittest.TestLoader().loadTestsFromTestCase(TestSequenceFunctions)!
\verb!unittest.TextTestRunner(verbosity=2).run(suite)! (as in the example on the website) to receive a method name and the status of the test. You could also use \verb5 if __name__ == '__main__':    unittest.main()5 but that only returns how many test cases passed and how many failed. I would strongly suggest we make the first option our \textbf{standard when writing test cases}.}
\item{To \textbf{run} the test case you can simply type \verb5python testcasename.py5 and voila!}
\end{itemize}
